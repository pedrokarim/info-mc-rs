Player Endpoints

These endpoints allow you to look up Minecraft player profiles and download their skin textures via the Mojang API.

Player Profile

GET /api/v1/player/{username}

Retrieves a player's profile information from the Mojang API, including their UUID, username, and skin/cape texture URLs.

Path Parameters

Parameter Type Description
username string Minecraft username (3–16 characters, alphanumeric and underscores) or UUID (with or without dashes).

Response

200 OK — Player Profile 
{
  "uuid": "069a79f4-44e9-4726-a5be-fca90e38aaf5",
  "username": "Notch",
  "skin": {
    "url": "http://textures.minecraft.net/texture/292009a4925b58f02c77dadc3ecef07ea4c7472f64e0fdc32ce5522489362680",
    "model": "classic"
  },
  "cape": {
    "url": "http://textures.minecraft.net/texture/afd553b39358a24edfe3b8a9a939fa5fa4faa4d9a9c3d6af8eafb377fa05c2bb"
  },
  "name_history": [
    {
      "name": "Notch",
      "changed_at": null
    }
  ],
  "cached": false,
  "timestamp": "2026-03-12T14:30:00Z"
}
Skin model: The model field is either "classic" (Steve, 4px arms) or "slim" (Alex, 3px arms). This is determined by the player's profile settings.

Errors

Status Code Description
400 invalid_address The username or UUID format is invalid.
404 player_not_found No player exists with the given username or UUID.
429 rate_limit_exceeded Too many requests. See Rate Limiting.

Example

curl http://localhost:3001/api/v1/player/Notch
curl http://localhost:3001/api/v1/player/069a79f4-44e9-4726-a5be-fca90e38aaf5

Raw Skin

GET /api/v1/player/{username}/skin

Downloads the raw skin texture PNG for a player. Returns the original Minecraft skin image as provided by Mojang.

Path Parameters

Parameter Type Description
username string Minecraft username (3–16 characters) or UUID.

Query Parameters

Parameter Type Default Description
download boolean false When true, sets Content-Disposition: attachment to trigger a file download in browsers.

Response

Returns the raw skin image with Content-Type: image/png. The image dimensions are 64x64 for modern skins or 64x32 for legacy skins.

Note: This endpoint returns a binary PNG image, not JSON. Error responses are still returned as JSON.

Example

# View in browser
curl http://localhost:3001/api/v1/player/Notch/skin -o notch_skin.png

# Force download with filename
curl "http://localhost:3001/api/v1/player/Notch/skin?download=true" -o notch_skin.png
