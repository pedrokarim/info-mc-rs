Server Endpoints

These endpoints allow you to ping Minecraft servers and retrieve their current status, MOTD, player counts, and historical data.

Ping Server

GET /api/v1/server/{address}

Pings a Minecraft server and returns its current status, including version, player count, MOTD, favicon, and latency.

Path Parameters

Parameter Type Description
address string Server IP address or domain name. Supports optional port (e.g., mc.example.com:25565). SRV records are resolved automatically.

Query Parameters

Parameter Type Default Description
type string auto Server type to ping. One of: java, bedrock, or auto (tries Java first, then Bedrock).
query boolean false When true, performs a full query (requires the server to have enable-query=true in its properties).

Response

200 OK — Server Online 
{
  "online": true,
  "host": "mc.hypixel.net",
  "port": 25565,
  "version": {
    "name": "1.8 - 1.21.4",
    "protocol": 47
  },
  "players": {
    "online": 42389,
    "max": 200000,
    "sample": [
      {
        "name": "Steve",
        "uuid": "8667ba71-b85a-4004-af54-457a9734eed7"
      }
    ]
  },
  "motd": {
    "raw": "\u00a76\u00a7lHypixel Network \u00a7c[1.8-1.21]",
    "clean": "Hypixel Network [1.8-1.21]",
    "html": "<span style=\"color:#FFAA00;font-weight:bold\">Hypixel Network</span> <span style=\"color:#FF5555\">[1.8-1.21]</span>"
  },
  "favicon": "data:image/png;base64,iVBORw0KGgo...",
  "latency": 34,
  "srv": {
    "host": "mc.hypixel.net",
    "port": 25565
  },
  "cached": false,
  "timestamp": "2026-03-12T14:30:00Z"
}
200 OK — Server Offline 
{
  "online": false,
  "host": "offline.example.com",
  "port": 25565,
  "error": "connection_refused",
  "cached": false,
  "timestamp": "2026-03-12T14:30:00Z"
}

Errors

Status Code Description
400 invalid_address The server address format is invalid.
422 dns_resolution_failed The domain could not be resolved via DNS.
429 rate_limit_exceeded Too many requests. See Rate Limiting.
504 timeout The server did not respond within the timeout period.

Example

curl http://localhost:3001/api/v1/server/mc.hypixel.net
curl "http://localhost:3001/api/v1/server/play.example.com:25565?type=java&query=true"

Server History

GET /api/v1/server/{address}/history

Returns historical player count data for a server over a specified time period. Data points are aggregated based on the chosen period.

Path Parameters

Parameter Type Description
address string Server IP address or domain name (same format as the ping endpoint).

Query Parameters

Parameter Type Default Description
period string 24h Time period for the history. One of: 1h, 6h, 24h, 7d, 30d.

Response

200 OK — History Data 
{
  "host": "mc.hypixel.net",
  "period": "24h",
  "data": [
    {
      "timestamp": "2026-03-11T14:00:00Z",
      "players_online": 38420,
      "online": true
    },
    {
      "timestamp": "2026-03-11T15:00:00Z",
      "players_online": 41205,
      "online": true
    },
    {
      "timestamp": "2026-03-11T16:00:00Z",
      "players_online": 43891,
      "online": true
    }
  ],
  "stats": {
    "min": 28103,
    "max": 48721,
    "avg": 39456
  }
}
Data resolution: The number of data points returned depends on the period. 1h returns per-minute data, 24h returns hourly data, and 30d returns daily aggregates.

Example

curl "http://localhost:3001/api/v1/server/mc.hypixel.net/history?period=7d"
