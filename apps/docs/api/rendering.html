Skin Rendering

The rendering endpoint generates 2D rendered images of Minecraft player skins. It supports multiple render types, custom sizes, and output formats.

Render Skin

GET /api/v1/render/{username}

Generates a 2D rendered image of a player's skin. The render is created server-side and returned as a PNG or WebP image.

Path Parameters

Parameter Type Description
username string Minecraft username (3–16 characters) or UUID.

Query Parameters

Parameter Type Default Description
type string bust Render type. One of: head (face only), bust (head + torso + arms), full (full body), face (flat face, no perspective).
size integer 256 Output image size in pixels. Range: 8 to 512. The image is always square.
overlay boolean true When true, renders the overlay layers (hat, jacket, sleeves, pants) on top of the base skin.
format string png Output image format. One of: png, webp.

Response

Returns the rendered skin image with the appropriate Content-Type header (image/png or image/webp). The image has a transparent background.

Performance: Rendered images are cached for 1 hour. Subsequent requests for the same player and parameters will be served from cache with minimal latency.

Render Types

Type Description Typical Use
face Flat front face of the skin, no 3D perspective Avatars, chat icons
head 3D isometric head render Profile pictures, leaderboards
bust Head, torso, and arms Player cards, social embeds
full Full body from head to feet Profile pages, galleries

Example URLs

# Default bust render (256px PNG)
curl http://localhost:3001/api/v1/render/Notch -o notch.png

# Full body, 512px
curl "http://localhost:3001/api/v1/render/Notch?type=full&size=512" -o notch_full.png

# Head only, 64px, WebP format
curl "http://localhost:3001/api/v1/render/Notch?type=head&size=64&format=webp" -o notch_head.webp

# Face without overlay layers
curl "http://localhost:3001/api/v1/render/Notch?type=face&size=128&overlay=false" -o notch_face.png

Usage in HTML

You can use render URLs directly in <img> tags:

<img src="http://localhost:3001/api/v1/render/Notch?type=head&size=64" alt="Notch">
<img src="http://localhost:3001/api/v1/render/Notch?type=full&size=256" alt="Notch full body">
Size limits: The minimum size is 8px and the maximum is 512px. Requests outside this range will return a 400 error.
