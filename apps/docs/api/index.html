API Overview

Base URL

http://localhost:3001/api/v1

The MCInfo API provides real-time data about Minecraft servers and players. All information is obtained directly from Minecraft servers via the native protocol and from the official Mojang API. No third-party data sources are used.

Response format: All endpoints return JSON responses, except for image endpoints (/skin and /render) which return PNG or WebP images directly.

Endpoint Overview

Method Endpoint Description
GET /api/v1/server/{address} Ping a Minecraft server and retrieve its current status
GET /api/v1/server/{address}/history Get historical player count data for a server
GET /api/v1/player/{username} Look up a player profile by username or UUID
GET /api/v1/player/{username}/skin Download the raw skin PNG for a player
GET /api/v1/render/{username} Get a rendered 2D image of a player's skin
GET /health Health check endpoint
GET /api/v1/stats Service statistics (uptime, cache info, request counts)

Authentication

The API is currently open and does not require authentication. Rate limiting is applied per IP address to prevent abuse. See Errors & Rate Limiting for details.

Error Handling

All error responses follow a consistent JSON structure. Refer to the Errors & Rate Limiting page for the full error format, error codes, and rate limit details.

{
  "error": {
    "code": "error_code_here",
    "message": "A human-readable description of the error."
  }
}
