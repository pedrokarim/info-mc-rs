MCInfo

Self-contained Rust platform for querying Minecraft servers, looking up players, and rendering skins in 2D & 3D.

Get Started API Reference
⚡

Direct Protocol

Pings Minecraft servers directly via SLP, Query, and Bedrock Raknet. No third-party APIs.

🎨

Skin Rendering

Server-side 2D isometric & 3D wgpu renders. Client-side Three.js interactive viewer.

🚀

Fast & Cached

Async Rust with Tokio & Axum. In-memory TTL caching via moka for blazing responses.

📡

REST API

Public API for servers, players, and skin rendering with rate limiting and structured errors.
